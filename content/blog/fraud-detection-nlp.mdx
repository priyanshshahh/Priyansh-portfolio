## The problem

Fraudulent job postings exploit text patterns and metadata inconsistencies. The challenge: high accuracy on imbalanced classification with interpretable, real-time scoring.

## The hybrid feature stack

Instead of deep learning alone, I engineered:

- **TF-IDF** vectorization for lexical fraud patterns
- **VADER sentiment** scores for emotional manipulation signals
- **Metadata features** from posting structure

<Callout title="Results" variant="results">
Multinomial Naive Bayes achieved 96.5% accuracy - a reminder that well-engineered classical features often outperform black-box models on structured text tasks.
</Callout>

## Deployment

A real-time **Dash dashboard** visualizes fraud probabilities per posting, making the model actionable for screening workflows.

## Takeaway

Feature engineering discipline matters more than model complexity for domain-specific text classification. Code at github.com/priyanshshahh/fraud-job-detection-pipeline.
