## Executive Summary

An end-to-end machine learning pipeline forecasting digital asset prices using LSTM neural networks, ARIMA, regularized regressions, and K-Means clustering with an interactive Tableau dashboard.

## Problem & Constraints

Cryptocurrency markets exhibit high volatility and non-stationarity, requiring comparison across multiple modeling paradigms with rigorous evaluation.

<Callout title="Architecture" variant="architecture">
REST API ingestion → preprocessing notebooks → regression + time-series + deep learning models → clustering analysis → Tableau interactive dashboard.
</Callout>

## Methodology

- Automated historical market data ingestion via REST APIs
- Built regression models, LSTM networks, and ARIMA time-series forecasts
- Applied K-Means clustering to uncover asset behavior patterns
- Deployed interactive Tableau dashboard for model performance comparison
