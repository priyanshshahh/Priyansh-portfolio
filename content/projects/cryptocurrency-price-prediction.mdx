## Executive Summary

An end-to-end machine learning pipeline forecasting digital asset prices using classical time series, deep learning, and clustering analysis with interactive Tableau visualization.

## Problem & Constraints

Cryptocurrency markets exhibit high volatility and non-stationarity, requiring comparison across multiple modeling paradigms with rigorous performance evaluation and visual analytics.

## Methodology

- Automated historical market data ingestion via REST APIs
- Implemented LSTM neural networks, ARIMA, and regularized regression models
- Applied K-Means clustering to uncover asset behavior patterns
- Deployed interactive Tableau dashboard for model performance comparison

## Results & Metrics

| Metric | Result |
|--------|--------|
| Models | LSTM, ARIMA, Regularized Regression |
| Clustering | K-Means asset segmentation |
| Visualization | Interactive Tableau dashboard |
| Data Pipeline | Automated REST API ingestion |

## Tech Stack

Python, Scikit-Learn, TensorFlow, Statsmodels, Tableau, REST APIs
