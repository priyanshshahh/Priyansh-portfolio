## Executive Summary

Equiply Enrichment is a React dashboard that ingests raw equipment data, normalizes fields through rule-based pipelines, and uses OpenAI API as an intelligent fallback for ambiguous or incomplete records.

## Problem & Constraints

Equipment datasets arrive in inconsistent formats with missing attributes. The system must balance automated normalization accuracy with cost-effective LLM enrichment for edge cases.

## Methodology

- Built React dashboard for data upload, review, and export workflows
- Implemented rule-based normalization pipeline for standard equipment fields
- Designed OpenAI fallback layer for records failing rule-based matching
- Added human-in-the-loop review queue for low-confidence enrichments
