## Executive Summary

A fraud detection system classifying job postings as legitimate or fraudulent using hybrid text and metadata features, deployed with a real-time Dash dashboard for probability visualization.

## Problem & Constraints

Online job platforms face growing scam volumes. The system must achieve high accuracy on imbalanced text classification while providing interpretable fraud scores.

<Callout title="Results" variant="results">
Achieved 96.5% classification accuracy using Multinomial Naive Bayes with hybrid TF-IDF and VADER sentiment features.
</Callout>

## Methodology

- Engineered hybrid TF-IDF vectorization combined with VADER sentiment scoring
- Trained Multinomial Naive Bayes with cross-validation on text + metadata features
- Deployed real-time Dash dashboard for fraud probability visualization

## Results & Metrics

| Metric | Result |
|--------|--------|
| Accuracy | 96.5% |
| Features | TF-IDF + VADER |
| Classifier | Multinomial Naive Bayes |
