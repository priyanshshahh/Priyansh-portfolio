## Executive Summary

PearPay is an agent payments and escrow platform developed during ETH Global, forked from mollybeach/pearpay with contributions for Flow integration, FaceID authentication, pay links, and Arc/Hedera escrow contracts. Live at [pearpay.vercel.app](https://pearpay.vercel.app/). Live at [pearpay.vercel.app](https://pearpay.vercel.app/).

## Problem & Constraints

Agent-to-agent and human-to-agent payments need secure, low-friction escrow with native mobile and web interfaces.

<Callout title="Architecture" variant="architecture">
Next.js web app + iOS iMessage/Apple Pay layer + Solidity escrow contracts (Arc + Hedera) + agent payment flows.
</Callout>

## Methodology

- Extended PearPay fork with Flow and FaceID payment features
- Built Arc and Hedera escrow smart contracts
- Implemented thin native iOS layer for iMessage and Apple Pay integration
- Added agent payment link generation and routing
