## Executive Summary

SHF-ML is a machine-learning empirical asset pricing study that measures forecast disagreement generated by different information sets. Inspired by the Machine Forecast Disagreement (MFD) literature and the Jensen-Kelly-Pedersen (JKP) characteristic dataset, the project simulates heterogeneous investor beliefs using two Random Forest forecasters - one trained on accounting features only, one on market features only.

## Problem & Constraints

The central research question: if two investors observe different categories of information, do they generate systematically different forecasts for next month's excess stock returns? The project runs on Seawulf and NVwulf HPC infrastructure with 10-year rolling windows and strict leakage-safe preprocessing. Code resides on university lab systems (not publicly distributed). References: [JKP Factors Database](https://jkpfactors.com/).

<Callout title="Architecture" variant="architecture">
JKP Monthly Dataset → feature partition (Accounting vs Market) → parallel Random Forest forecasters → disagreement measure (Prediction_A - Prediction_M) → HPC batch execution on SeaWulf.
</Callout>

## Methodology

- Used JKP Monthly Dataset (Jensen-Kelly-Pedersen Global Factor Database, 2015-2025)
- Partitioned 153 stock characteristics into accounting-only and market-only subsets
- Trained separate Random Forest models targeting `ret_exc_lead1m`
- Computed forecast disagreement as the difference between accounting and market predictions
- Deployed reproducible pipeline on Seawulf/NVwulf with partitioned batch jobs
